# Collection Search
##### An app written in `react` for keeping a collection of whatever! The app is demoed here for a collection of articles. 

## Tools:
* react and react-dom

## How it's been written?
* simple conditional rendering with no routing
* simple CSS styling
* no redux
* no type checking

## What's next?
* Routing + Redux
* Type checking (`flow` maybe)
* Linting
* Adding a simple backend

## App features:
* List your items
* Search through
* Add a new one
* Edit an existing one
* Delete one
* Clap an item

## How to run?
* npm install
* npm run
* localhost:3000
